# Project Images

## Required Images

### 1. Logo (logo.png)
- **Source**: First image from conversation (head with leaf icon in blue circle)
- **Dimensions**: 120x120px recommended
- **Format**: PNG with transparent background
- **Usage**: Navigation header across all pages

### 2. Hero Background (hero-bg.jpg)
- **Source**: Second image from conversation (mental health illustration with person)
- **Dimensions**: 1920x1080px or larger
- **Format**: JPG or PNG
- **Usage**: Homepage hero section background

## How to Add Images

1. Right-click on each image from the conversation
2. Save them to this directory with the filenames above
3. Refresh your browser to see the changes
